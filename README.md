# System Monitor CLI

A Java command-line tool that displays system information and exports a system report.

## Features

- System Information (OS, user, Java version)
- CPU Information
- Memory Information
- Network Information
- Disk Information
- Export system report to file

## How to Run

Compile:

javac -d . src/com/system/monitor/*.java

Run:

java com.system.monitor.Main
